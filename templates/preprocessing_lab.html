Preprocessing Lab

Compare two preprocessing pipelines side-by-side (Tesseract only)

Option 1 — Grayscale Pipeline + Background Flatten (tunable)

Upscale → Grayscale → Gaussian denoise → Background flatten → Normalize → Unsharp mask → Deskew. Stays grayscale (no binarization). Hover a label for details.

Option 2 — CLAHE + Adaptive Threshold + Morph Line Removal

Upscale → Grayscale → CLAHE → Adaptive Gaussian Threshold → Morph line removal. Hover a label for details.

Option 3 — Background-flattened + Adaptive Threshold + Stroke Closing

Upscale → Grayscale → Background flatten (divide by MORPH_CLOSE estimate) → CLAHE → Adaptive threshold → Stroke closing → Optional line removal. Fixes Option 2's watermark bleed and broken strokes.
